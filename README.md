# minble - vue.js
1인 프로젝트 <br>
vus.js 로는 처음 개발해보았습니다. <br><br>

SNS 형식의 팬 카페입니다. vue.js 부트스트랩을 사용하여 개발하였습니다. 부트스트랩을 사용하면서 필요한 폴더와 파일을 추가하며 사용했기 때문에 형식이 깔끔하지 않습니다. 추후 지속적으로 수정과 삭제를 해 나아갈 예정입니다.
